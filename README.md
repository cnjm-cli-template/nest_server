## 演示

本项目的构建过程记录在[nest.js](https://blog.csdn.net/weixin_42998707/category_11219947.html)专栏，希望对你有用

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
